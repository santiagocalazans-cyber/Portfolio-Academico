# Portfolio-Academico
Um repositório criado para objetivos acadêmicos e profissionais, com informações pessoais do autor.
No arquivo "documentacao" possui o currículo e dados importante. 
No arquivo "projetos-academicos" possuem os projetos feitas na instituição a qual estudo.
No arquivo "projetos-pessoais" tem os projetos em estou trabalhando, ou já realizei.
